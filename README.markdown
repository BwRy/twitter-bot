## Description
Shell script to download music from http://www.soundcloud.com.
Supports multiple pages.

## System requirements
* Unix like OS (Linux, Mac OS X...)
* curl

## Instructions
* apply executable permissions (chmod +x ./tweet.sh)
* usage: ```./tweet.sh "Status"```
